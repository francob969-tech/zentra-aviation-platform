## Título del gig

**No vendo páginas. Construyo sistemas que convierten visitantes en clientes.**

Soy ZENTRA Studio. Diseño y desarrollo plataformas web premium para operadores

Mi último proyecto: una **plataforma completa de captación de clientes para un
operador de taxi aéreo privado**. No es una landing más; es un sistema comercial.

**Qué incluye:**
- 🎯 **Sitio premium** con estética de aviación ejecutiva (no plantillas, no stock).
- 📩 **Captación de leads de doble registro:** cada consulta se abre en WhatsApp
  Y queda guardada en el sistema, aunque el cliente no concrete el envío.

- 🔍 **Arquitectura SEO completa** (metadata, datos estructurados, Open Graph, sitemap).

**Tecnología:** Next.js, React, TypeScript, Tailwind, Supabase. Listo para
producción, rápido, responsive y medible.
