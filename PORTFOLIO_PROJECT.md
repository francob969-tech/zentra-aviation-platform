# 🇪🇸 Versión en español

## Resumen
Sistema digital completo para un operador de taxi aéreo privado: un sitio web
premium que no solo comunica el servicio, sino que **convierte visitantes en
solicitudes de vuelo registradas y gestionables** desde un panel propio. La web
es la cara visible; debajo opera un sistema de captación y seguimiento de leads.

| | |
|---|---|
| **Industria** | Aviación privada / charter regional |
| **Tipo de proyecto** | Sitio web premium + captación de leads + panel de gestión (CRM) |
| **Alcance** | Estrategia · UX/UI · Desarrollo · Backend · Panel administrativo |
| **Stack** | Next.js · React · TypeScript · Tailwind · Supabase |

## El desafío de negocio
Un operador de aviación privada que inicia no tiene un problema de publicidad,
sino de **confianza y conversión**:
- Las consultas llegaban por canales sueltos (WhatsApp, llamados, contactos) y
  se perdían sin registro ni seguimiento.
- No existía una presencia digital a la altura del nivel premium que el cliente
  ejecutivo espera.
- El valor real del servicio —ahorro de tiempo— no estaba articulado en ningún
  lado.
- En un sector regulado, comunicar de más es un riesgo: hacía falta un mensaje
  potente pero prudente.

## Enfoque de diseño
Estética de aviación ejecutiva: navy profundo, plata, acentos cian, grilla de
radar y líneas finas. Tipografía editorial y microinteracciones sutiles —premium
real, sin recursos genéricos ni imágenes de stock—. Cada sección está construida
para guiar hacia una sola acción: solicitar una cotización.

## Estrategia de captación de leads
El núcleo es una arquitectura de **doble registro**: cuando un visitante solicita
cotización, el pedido se abre en WhatsApp **y** se guarda en el backend de forma
simultánea. Así, aunque el visitante no concrete el envío, **la solicitud queda
registrada**. Validación de datos, protección anti-bots y limitación de
frecuencia completan una entrada robusta.

## Integración con WhatsApp
Múltiples puntos de contacto no invasivos (botón flotante, portada, formulario)
abren WhatsApp con el mensaje pre-armado —origen, destino, fecha, pasajeros—.
Usa el WhatsApp habitual del operador, sin costos de API. Reduce la fricción al
mínimo: el cliente puede consultar en un toque.

## Panel administrativo (CRM)
Un panel privado y protegido convierte al sitio en herramienta de trabajo:
bandeja de solicitudes, pipeline de estados (nuevo → contactado → cotizado →
ganado / perdido), notas internas, respuesta por WhatsApp con un clic y métricas
rápidas. Incluye **auto-actualización**: las consultas nuevas aparecen y se
destacan solas, sin recargar.

## Presentación de la aeronave
Página dedicada al Cessna 402 con overview, experiencia a bordo, perfil
operativo y lenguaje de seguridad. Redacción cauta y profesional, sin afirmar
cifras de rendimiento ni certificaciones no confirmadas —apropiado para un
sector regulado—.

## Visualización de rutas
Mapa animado tipo radar que posiciona la base de operaciones y traza las
conexiones regionales con líneas que se dibujan al hacer scroll. Comunica
cobertura de un vistazo, sin depender de mapas de terceros.

## Calculadora de tiempo ahorrado
Herramienta interactiva que compara, puerta a puerta, el traslado por auto, por
aerolínea comercial y por taxi aéreo según el destino. Traduce el servicio a su
beneficio concreto —horas recuperadas— con valores estimados y prudentes.

## Arquitectura SEO
Metadata completa, datos estructurados (JSON-LD), imagen Open Graph generada,
sitemap, robots y favicon. Renderizado estático para carga veloz y páginas
indexables. Eventos de conversión instrumentados para medir el retorno de
cualquier campaña.

## Resultado
Un sistema comercial completo, listo para producción, que captura, registra y
gestiona clientes con una estética a la altura de la aviación privada.

# 🇬🇧 English version

## Overview
A complete digital system for a private air taxi operator: a premium website
that doesn't just present the service but **turns visitors into recorded,
manageable flight requests** through a dedicated admin panel. The site is the
visible surface; underneath runs a lead capture and follow-up system.

| | |
|---|---|
| **Industry** | Private aviation / regional charter |
| **Project type** | Premium website + lead capture + management panel (CRM) |
| **Scope** | Strategy · UX/UI · Development · Backend · Admin panel |
| **Stack** | Next.js · React · TypeScript · Tailwind · Supabase |

## The business challenge
A private aviation operator starting out doesn't have an advertising problem —
it's a **trust and conversion** problem:
- Inquiries arrived through scattered channels (WhatsApp, calls, referrals) and
  were lost without record or follow-up.
- There was no digital presence matching the premium standard executive clients
  expect.
- The real value of the service — time saved — wasn't articulated anywhere.
- In a regulated sector, overstating is a risk: the messaging had to be strong
  yet prudent.

## Design approach
Executive aviation aesthetic: deep navy, silver, cyan accents, radar grid and
thin lines. Editorial typography and subtle microinteractions — genuinely
premium, no generic assets or stock imagery. Every section is built to guide
toward a single action: requesting a quote.

## Lead generation strategy
The core is a **dual-capture** architecture: when a visitor requests a quote,
the request opens in WhatsApp **and** is saved to the backend simultaneously. So
even if the visitor never sends the message, **the request is recorded**. Input
validation, anti-bot protection and rate limiting complete a robust entry point.

## WhatsApp integration
Multiple non-intrusive touchpoints (floating button, hero, form) open WhatsApp
with a pre-filled message — origin, destination, date, passengers. It uses the
operator's regular WhatsApp, with no API costs. Friction is minimal: a client
can inquire in one tap.

## Admin panel (CRM)
A private, protected panel turns the site into a working tool: request inbox,
status pipeline (new → contacted → quoted → won / lost), internal notes,
one-click WhatsApp reply and quick metrics. It includes **auto-refresh**: new
inquiries appear and are highlighted on their own, no reloading.

## Aircraft presentation
A dedicated Cessna 402 page covering overview, onboard experience, operational
profile and safety language. Cautious, professional copy that avoids stating
performance figures or unconfirmed certifications — appropriate for a regulated
sector.

## Route visualization
An animated radar-style map that anchors the home base and traces regional
connections with lines that draw in on scroll. It communicates coverage at a
glance, with no dependency on third-party maps.

## Time-saved calculator
An interactive tool that compares, door to door, travel by car, by commercial
airline and by air taxi for each destination. It translates the service into its
concrete benefit — hours recovered — using estimated, prudent values.

## SEO architecture
Complete metadata, structured data (JSON-LD), generated Open Graph image,
sitemap, robots and favicon. Static rendering for fast loads and indexable
pages. Conversion events instrumented to measure the return on any campaign.

## Outcome
A complete, production-ready commercial system that captures, records and
manages clients with an aesthetic worthy of private aviation.
